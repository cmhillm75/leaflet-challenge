# leaflet-challenge
Module 15 homework - Create an earthquake visualization using USGS data and open source maps.

## Prerequisites

- Web browser
- Internet connection
- VS Code

## Data Source
[GeoJSON](https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all_week.geojson)
